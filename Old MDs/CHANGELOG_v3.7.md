# Changelog - Version 3.7 (Build 32)

- **Draft Advisor** — Smart pick recommendations based on roster needs, ADP value, positional scarcity, and stat lines
- **Player Detail Cards** — Tap any player to see full stats, injury status, ESPN/depth chart links, and advisor score
- **Configurable Keeper Rounds** — Set 0-10 forced linear rounds for keeper leagues (replaces old checkbox)
- **Sort by ADP or Rank** — New config option to sort players by ADP instead of overall rank
- **Condensed Config** — Teams, SMS, position requirements, and player data moved to sub-page dialogs
- **Help & Guide** — In-app documentation for grading system, draft screen, and all features
- **Bug Fixes** — Corrected pick value calculation, position requirements persistence, dark mode visibility, K/DST excluded from grading
